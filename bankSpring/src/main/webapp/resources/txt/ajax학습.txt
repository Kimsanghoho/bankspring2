AJAX (Asynchronous Javascript And XML)

AJAX 는 새로운 프로그램 언어가 아니라 기존에 있던 기술들을 
조합하여 새로운 방법을 만들어 낸 방법론적인 기술이다.

조합된 언어
1. HTML (컨텐츠)
2. CSS (스타일)
3. Javascript, DOM (동적인 출력과 상호작용)
4. XML (데이터 전송)
5. XMLHttpRequest (비동기적 데이터 교환)

AJAX 의 필요성
기존의 웹브라우저는 서버로부터 페이지 단위로만 받을 수 있었다.
때문에 간단한 상호작용을 하려고 해도 페이지 전체를 로딩해야했고,
이것은 대역폭 낭비를 초래했다.
하지만 AJAX 를 사용하면 XML 파일 조각 단위로 데이터를 받아 올 수
있기 때문에 전체 페이지를 다시 로드하지 않아도 페이지 일부를
갱신할 수 있다.

AJAX 방법 : 총 5가지만 존재함
1. $.post() 방식
2. $.get() 방식
3. $.getJSON() 방식
4. $.ajax() 방식
5. $.load() 방식
